# Pricing Freshness

## Rule

Do not publish exact public prices unless Victor confirms them directly for the current season.

## Safe public wording

- Use `ask Victor for today's price and sea conditions` on public pages.
- Keep payment wording broad unless Victor asks for something more specific.
- If a page needs a stronger CTA, mention `WhatsApp Victor for current price, group fit, and sea conditions`.

## What needs a human check

- Current adult / child pricing
- Any resident / foreigner split
- Group-size minimums
- Cash / Bitcoin wording if Victor changes preference
